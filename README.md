Lyricist
======

Lyricist displays the lyrics of the currently playing song in [Banshee](http://en.wikipedia.org/wiki/Banshee_%28media_player%29). 

News
----

**05.20.14** - A functional version of lyricist now exists.

Sources
-------
AZLyrics.com
SongLyrics.com
